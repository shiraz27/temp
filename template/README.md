# React native boilerplate

** Libraries **

- Navigation (bottom, drawer)
- Dark Mode (theme) toggler
- Language localization

** UI **

- Header Component
- ScreenContainer Component
- Toast
- Ui kitten
- Infinit Scroll Flatlist
- Refresh Data Flatlist

** API & DATA **

- redux / asyncstorage user management
- Login
- Logout

## Getting started

- npm i
- cd ios && pod install
- npm run ios
